Digital Cleansing/数据清道夫

项目简介
    《数据清道夫》是一款由DoorBell Labs工作组制作的基于网格2D动作游戏，使用Unity引擎开发（版本号2022.3.62f3c1）。
     玩家扮演一个超级数据网络中的杀毒软件，深入网络内部，破坏防火墙以深入网络核心，最终击败藏匿于根服务器的病毒母体，并顺手消灭路上的病毒。游戏为半线性流程，包含3条支线共9个关卡，以及最终的boss战。

使用说明
    本项目支持Windows操作系统。
    下载仓库中Digital Cleansing文件夹，运行其中的.exe文件。或将仓库整个下载到本地，并用对应版本Unity引擎打开文件夹。
    游玩时请切换至英文输入法。
    移动/攻击 ———— WASD/方向键
    暂停 ———— Esc键
    重开 ———— R
    界面交互 ———— 鼠标左键单击按钮
